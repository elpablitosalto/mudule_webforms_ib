# wpg.webforms

Модуль для размещения веб-форм на сайте и сохранения заявок в инфоблок.

## Компонент

`wpg:webform` — выводит форму по свойствам ИБ, валидирует и сохраняет заявку как элемент ИБ.

### Основные параметры

- `IBLOCK_ID`: ID или CODE инфоблока для заявок
- `PROPERTY_CODES`: коды свойств (через запятую), которые выводить/сохранять
- `REQUIRED_PROPERTY_CODES`: обязательные коды (через запятую)
- `ELEMENT_NAME_TEMPLATE`: шаблон имени элемента (например `Заявка от #DATETIME#`)
- `ACTIVE_AFTER_SUBMIT`: `Y/N` — активировать элемент после отправки
- `SUCCESS_MESSAGE`: текст при успешной отправке
- `FORM_ID`: HTML id формы (для нескольких форм на странице)

## Установка

Модуль устанавливается из админки Битрикс. Компонент копируется в `local/components/wpg/webform`.
